# AI Conversation Cleaner
多平台AI对话批量清理脚本集合，**纯浏览器本地运行**，自动批量删除历史对话，保护隐私。

## 支持平台
- 豆包 Doubao
- Kimi
- ChatGPT
- 通义千问
- DeepSeek
- 持续更新中...

## 核心特性
- 纯前端浏览器脚本，**不上传任何个人数据**
- 自动分批+延迟，避免接口风控
- 无隐私收集、无第三方上报
- 所有ID使用随机UUID，不携带个人标识
- 可单独运行单个平台脚本

## 使用通用步骤
1. 打开对应AI网页版，登录账号
2. 按 `F12` 打开开发者工具 → Console 控制台
3. 复制对应平台文件夹内的脚本粘贴运行
4. 自动批量删除，等待控制台提示完成

## 注意事项
- 删除操作**不可逆**，请谨慎执行
- 脚本仅调用官方公开接口
- 所有操作在本地浏览器完成，不泄露隐私
- 速率可自行修改 `BATCH`（批量数）、`PAUSE`（间隔毫秒）
